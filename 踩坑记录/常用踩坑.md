## Git 分支分叉（diverged）导致 push 失败

### 现象
```
! [rejected] main -> main (fetch first)
error: failed to push some refs to 'github.com:jinlong85/obsidian-vault.git'
hint: Updates were rejected because the remote contains work that you do not
      have locally.
```

### 解决
**保留本地版本**：
```bash
git reset --hard origin/main && git push --force
```

**保留远程版本**：
```bash
git pull origin main --no-rebase && git push
```

## 文件显示 modified 但无实质内容变化

### 现象
`git status` 显示文件 modified，但 `git diff` 显示 0 行改动。

### 原因
- **换行符差异**：CRLF (`\r\n`) vs LF (`\n`)
- **文件权限差异**：100644 vs 100755

### 解决
无实质影响，可忽略。若需消除：
```bash
git diff --stat  # 确认是 0 改动
```

## Obsidian Git 插件 SSH 认证失败

### 现象
```
Host key verification failed
Permission denied (publickey)
Could not read from remote repository
```

### 原因
Windows 缺少 SSH 私钥或未配置 SSH config。

### 解决

1. **复制私钥到 Windows**：
   ```bash
   # 在 WSL 中执行
   cp ~/.ssh/id_ed25519 /mnt/c/Users/JINLONG/.ssh/
   ```

2. **创建 SSH config** (`C:\Users\JINLONG\.ssh\config`)：
   ```
   Host github.com
       HostName github.com
       User git
       IdentityFile ~/.ssh/id_ed25519
       AddKeysToAgent yes
   ```

3. **验证连接**：
   ```bash
   ssh -T git@github.com
   # 期望：Hi jinlong85! You've successfully authenticated...
   ```

## Windows Git 代理问题

### 现象
```
Failed to connect to github.com port 443
```

### 解决
```powershell
git config --global http.proxy http://127.0.0.1:9999
git config --global https.proxy http://127.0.0.1:9999
```

### 清除代理
```powershell
git config --global --unset http.proxy
git config --global --unset https.proxy
```


## npm install 失败

### 解决
```bash
# 使用代理
npm install --proxy http://127.0.0.1:9999

# 清理缓存重试
npm cache clean --force
rm -rf node_modules
npm install
```

### 国内包绕过代理
```bash
PROXY_CN=direct npm install
```

## WSL 与 Windows 路径问题

- WSL 路径：`/home/jinlong/...`
- Windows 路径：`C:\Users\...`
- WSL 访问 Windows：`/mnt/c/...`
- 不要混用，记住各自的路径格式。

## Obsidian Git 插件手动安装

### 问题
社区插件市场搜索不到。

### 解决
手动下载到 `.obsidian\plugins\obsidian-git\`：
```powershell
cd "C:\Users\JINLONG\obsidian-vault\.obsidian\plugins\obsidian-git"
curl -Proxy "http://127.0.0.1:9999" -o manifest.json "https://github.com/denolehov/obsidian-git/releases/download/2.5.0/manifest.json"
curl -Proxy "http://127.0.0.1:9999" -o main.js "https://github.com/denolehov/obsidian-git/releases/download/2.5.0/main.js"
curl -Proxy "http://127.0.0.1:9999" -o styles.css "https://github.com/denolehov/obsidian-git/releases/download/2.5.0/styles.css"
```

## Obsidian Git 插件必须配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| `autoPullOnBoot` | `true` | 启动时自动拉取 |
| `differentIntervalCommitAndPush` | `false` | 立即推送 |
| `autoSaveInterval` | `1` | 1分钟自动备份 |
| `autoPushInterval` | `2` | 2分钟自动推送 |
